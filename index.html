ULTIMATE TIC-TAC-TOE

Welcome to Ultimate Tic-Tac-toe! The aim of the game is to secure your box in the mini game to make sure you win the main game! Each player will take it in turns to place an X or O in any given box. Once a player has won that box it becomes there’s. if it’s a draw then the box will turn red and uncalmable. Remember that you don’t have to stick to one mini game at a time, so it can get a bit confusing!
